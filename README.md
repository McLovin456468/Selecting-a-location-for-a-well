# Проект: Выбор региона для бурения новой скважины

## 1. Назначение и цель проекта

Определить регион РФ, где бурение новой скважины принесёт максимальную прибыль при допустимом уровне риска.  
На основе исторических данных геологоразведки требуется построить модель линейной регрессии, оценить прибыль и риски методом Bootstrap, выбрать регион для разработки.

## 2. Исходные данные

Три файла с данными геологоразведки:

| Файл | Описание |
|------|----------|
| `geo_data_0.csv` | Данные по региону 0 |
| `geo_data_1.csv` | Данные по региону 1 |
| `geo_data_2.csv` | Данные по региону 2 |

### Структура каждого файла

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | строка | уникальный идентификатор скважины |
| `f0`, `f1`, `f2` | число | три значимых признака (содержание не раскрывается) |
| `product` | число | объём запасов в скважине (тыс. баррелей) |

## 3. Условия и ограничения бизнеса

- Для обучения и прогнозирования используется **только линейная регрессия** (остальные модели считаются недостаточно предсказуемыми).
- В каждом регионе исследуется **500 точек** (скважин).
- Для разработки отбираются **200 лучших** скважин по предсказанному объёму запасов.
- Бюджет на разработку скважин в одном регионе: **10 млрд рублей**.
- Доход с 1 барреля сырья: **450 рублей**.
- Объём запасов указан в **тысячах баррелей**, поэтому доход с единицы продукта = **450 тыс. рублей**.
- Допустимый уровень риска убытков (отрицательной прибыли): **не более 2.5%**.
- Выбор региона производится среди тех, где риск убытков < 2.5%, по критерию **максимальной средней прибыли**.

## 4. Этапы выполнения

### 4.1. Загрузка и подготовка данных
- Загрузить три датасета.
- Проверить пропуски, типы данных, явные и неявные дубликаты.
- Привести данные к единому формату.
- *Примечание:* синтетический характер данных не требует глубокой EDA по физическому смыслу признаков.

### 4.2. Обучение и проверка модели (для каждого региона)
- Разделить данные на обучающую (75%) и валидационную (25%) выборки.
- Обучить модель линейной регрессии на обучающей выборке.
- Выполнить предсказания на валидационной выборке.
- Сохранить предсказания и фактические значения.
- Вычислить и вывести:
  - средний предсказанный объём запасов
  - RMSE модели
- Проанализировать качество модели по каждому региону.

### 4.3. Подготовка к расчёту прибыли
- Сохранить ключевые параметры в переменные:
  - бюджет на разработку
  - доход на 1 тыс. баррелей
  - количество исследуемых точек
  - количество отбираемых скважин
- Рассчитать **безубыточный объём сырья** (в тыс. баррелей):
  - объём, при котором доход от 200 скважин покрывает бюджет.
- Сравнить безубыточный объём со средними запасами в каждом регионе (фактическими и предсказанными).
- Сделать вывод о потенциальной пригодности регионов.

### 4.4. Функция расчёта прибыли
Разработать функцию, которая принимает:
- предсказания модели
- фактические значения продукта
- количество отбираемых скважин (N = 200)

**Логика функции:**
1. Отобрать N скважин с наибольшими предсказанными значениями.
2. Взять соответствующие им фактические объёмы запасов.
3. Просуммировать фактические объёмы.
4. Рассчитать прибыль:
   - доход = сумма_фактических_запасов × доход_на_1_тыс_баррелей
   - прибыль = доход − бюджет

### 4.5. Оценка рисков и прибыли (Bootstrap)
Для каждого региона выполнить:

- **1000 итераций бутстрепа**:
  - из валидационной выборки случайно с выбором возвратом выбрать выборку размера = валидационная выборка
  - обучить модель (опционально — использовать фиксированную модель, применённую к бутстреп-выборке)
  - для каждого бутстреп-образца рассчитать прибыль по 200 лучшим скважинам
- По полученному распределению прибыли рассчитать:
  - среднюю прибыль
  - 95% доверительный интервал
  - риск убытков (доля итераций с отрицательной прибылью)

### 4.6. Выбор региона
- Отфильтровать регионы с риском убытков < 2.5%
- Среди оставшихся выбрать регион с максимальной средней прибылью
- Обосновать выбор

## 5. Выходные результаты

- Метрики качества модели (средний предсказанный запас, RMSE) для трёх регионов.
- Расчёт безубыточного объёма и его сравнение с фактическими/предсказанными средними запасами.
- Распределение прибыли по каждому региону (среднее, доверительный интервал, риск убытков).
- Итоговое решение: выбранный регион с обоснованием.

## 6. Критерии приёмки

- Модель — только линейная регрессия.
- Учтены все бизнес-условия (200 скважин, бюджет 10 млрд, доход 450 руб/баррель).
- Bootstrap выполнен корректно (1000 выборок, расчёт доверительного интервала и риска).
- Вывод содержит конкретный регион и развёрнутое обоснование.
- Код не дублируется, логика вынесена в функции.
